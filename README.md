# DreamJob

Учебный веб-проект на Java с использованием Spring Boot и Thymeleaf.

## Технологии
- Java 21+
- Spring Boot
- Spring MVC
- Thymeleaf
- Maven
- Bootstrap

## Описание
Платформа для поиска работы:

- кандидаты публикуют резюме
- работодатели размещают вакансии
- взаимодействие между пользователями через веб-интерфейс
